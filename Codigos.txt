*CRIAÇÃO DE TABELHA

Dentro da TAG BODY, criar a seguinte tag:

<table border=""></table> = tabela

border = para escolher tamanho da borda

DENTRO DE TABLE

<tr></tr> = table row = linha da tabela

DENTRO DE TR 

<td></td> = table date = dados da tabela
	
	Dentro do <td>, é possível adicionar um atributo para agrupar mais de uma coluna
	<td colspan=""> extensão da coluna
	
	Dentro do <td>, é possível adicionar um atributo para agrupar mais de uma linha
	<td rowspan=""> extensão da linha

<th></th> = Table Header = cabeçalho da tabela
